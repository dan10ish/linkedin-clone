# LinkedIn Clone 

<br>

A fully functional LinkedIn clone made using

* `React` (FrontEnd)
* `Redux`
* `Material UI`
* `Firebase` (BackEnd)

<br>

---
You can view this web app in real-time [here](https://linkedin-clone-667e9.web.app/).
